Toepassingslaag

De toepasingslaag is een van de drie lagen van het drielagenmodel. Het valt onder het begrip 'software'. De toepasingslaag bestaat onder andere uit apps. Een app is een klein progamma op je telefoon. Apps kunnen we onderscheiden in gedownloade apps zoals WhatsApp, spotify en instagram en in systeemapps zoals google of safari. Apps zorgen ervoor dat gebruikers hun behoefte kunnen vervullen, zoals informatie opzoeken, communiceren en online aankopen doen. Daarnaast heb je bij de toepaingslaag ook nog API's. API staat voor: 'Application Programming Interface'. Een API maakt communicatie mogelijk tussen verschillende (software) applicaties/programma's. Een API zorgt er dus voor dat apps informatie met elkaar kunnen uitwisslen met behulp van een server. Het maakt dataoverdrachten van systeem naar systeem mogelijk. Een voorbeeld hierbij: stel je hebt een app open met daarin een artikel over Londen. Je wilt graag weten hoe koud/warm het is in Londen. Hierbij kan een andere app bijvoorbeeld de app 'weerbericht' informatie uitwisselen en overdragen naar de app met de artikel.

In dit filmpje wordt API duidelijk uitgelegd!
